# Image Assets

This folder contains image assets for the Pomodoro Timer web application.

## Current Images

- `placeholder-about.svg` - Placeholder illustration for the About page
- `favicon.svg` - Website favicon (tomato icon)
- `hero-illustration.svg` - Hero section illustration

## Adding Custom Images

Replace these placeholder SVG files with your own images:

1. **Logo**: Add a logo file (e.g., `logo.png` or `logo.svg`) for the navigation
2. **Hero Images**: Add background images or illustrations for the homepage
3. **Feature Icons**: Add custom icons for the features section
4. **About Page**: Replace `placeholder-about.svg` with a proper illustration
5. **Favicon**: Replace `favicon.svg` with your preferred icon

## Recommended Formats

- **Icons/Logos**: SVG or PNG with transparency
- **Photos/Illustrations**: JPG or PNG
- **Favicon**: ICO, SVG, or PNG (32x32 or 16x16)

## Optimization

Remember to optimize images before adding them:
- Compress JPG/PNG files
- Minify SVG files
- Use appropriate dimensions for web display
